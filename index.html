Datasets of Interest from awesome-datascience github:

The data set of interest that I chose ultmately came from a Kaggle inventory which held a data set describing data science job salaries. This data set contains features/variables including: work_year, experience_level, job_title, salary, company_size, company_location etc. 'Data science jobs' encompass a very large field with a lot niches and varying responsabilities. This data set is relevant to our class, just as well for anyone looking into the different avenues of data science. The relative compensation will grant a decent window into the financial endowments that accompany a data science opportunity of a particular profession, applicant type, relative experience, regional selectivity, etc.
